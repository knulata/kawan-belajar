Panduan Setup iPad Kiosk

Setup iPad untuk Tuition Center

1

Install Kawabel Web App

1. Buka Safari di iPad

2. Kunjungi URL Kawabel:

3. Tap tombol Share (kotak dengan panah ke atas)

4. Pilih "Add to Home Screen"

5. Beri nama "Kawabel" lalu tap Add

2

Aktifkan Guided Access (Kiosk Mode)

Guided Access mengunci iPad ke satu app saja — siswa tidak bisa keluar.

  1. Buka Settings > Accessibility > Guided Access
  2. Toggle ON
  3. Set Passcode (kode untuk guru keluar dari kiosk mode)
  4. Enable Time Limits > Speak (optional, untuk pengingat waktu)
  5. Buka app Kawabel dari Home Screen
  6. Triple-click tombol Home/Side untuk mulai Guided Access
  7. Tap Start di kanan atas

Untuk keluar: Triple-click tombol Home/Side, masukkan passcode guru

3

Setting Keamanan Tambahan

  1. Settings > Screen Time > Content & Privacy — block Safari, App Store, Settings changes
  2. Settings > General > Auto-Lock — set ke "Never" (iPad selalu nyala)
  3. Settings > Display > Brightness — auto-brightness ON untuk hemat baterai
  4. Settings > Wi-Fi — auto-join Wi-Fi tuition center
4

Buat PIN untuk Setiap Siswa

1. Buka Halaman Siswa di admin dashboard

2. Tambah siswa baru — PIN otomatis dibuat (4 digit)

3. Print kartu PIN untuk setiap siswa

4. Siswa login dengan PIN di iPad — auto-logout setelah 5 menit idle

Troubleshooting

App tidak mau loading?

Cek koneksi Wi-Fi. Restart app. Clear browser cache jika perlu.

Siswa tidak bisa login?

Cek PIN di halaman Siswa. Reset PIN jika perlu.

Kawi tidak merespon?

Server mungkin down. Cek /api/health. Siswa tetap bisa latihan dari bank soal offline.

Cara keluar Guided Access?

Triple-click tombol Home/Side button, masukkan passcode guru.
